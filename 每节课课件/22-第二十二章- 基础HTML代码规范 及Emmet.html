22-第二十二章- 基础HTML代码规范 及Emmet

一、HTML规范

1、 HTML代码书写规范

1）所有书写均在英文半角状态下的小写；
2）所有标签必须闭合；
3）统一用tab键缩进；
4）属性值必须带引号（单引或双引）；
5）ul,li/ol,li/dl,dt,dd拥有父子级关系的标签；
6）p,dt,h标签 里面不能嵌套块属性标签；
7）a标签不能嵌套a；
8）内联元素不能嵌套块；（a除外）
9）对“<”“>”之类的特殊符号进行实体转义`<>

2、HTML注释

1、单行注释：<!-- 注释内容 -->
2、多行注释：

1.<!-- 

2.    注释内容一

3.    注释内容二

4.    注释内容三

5.-->

1.<!-- 

2.*   注释内容一

3.*   注释内容二

4.*   注释内容三

5.-->

3、 HTML命名规范

1）命名的两种方式：

ID命名： 具有唯一性(身份证号) 相同的ID名只能在页面中出现一次， 并且ID只能有一个名字
class类名：class的类名可以在页面中重复出现，并且一个元素可以有多个类名,(中间以空格分开)

2）HTML命名规范：

A 用小写字母
B 以英文开头 可以包含（英文字母 - _ 数字）
C （见名知意）

3）拓展名写法:

A 加 - 或 _
（腾讯：下划线 ’ _ ’ 禁止出现在class命名中，统一使用 ‘-’ 连字符）
B 驼峰命名 第二单词大写
（腾讯：禁止驼峰式命名）

5、图片

1）所有img元素必须加上width height alt属性,修饰性图片alt值留空
2）图片命名

  • A) 图片后缀命名一律小写。
  • B) 使用间隔符 - 进行连接。
    • 一般背景图片以bg-开头，
    • 按钮图片以btn-开头，
    • 图标图片以icon-开头，
    • 聚合图以spr-开头，后跟英文单词，如果名称过长，适当使用缩写

6、文件命名

1）文件名中只可由英文字母a~z、排序数字0~9或间隔符-组成。
2）蚊件名中禁止包含特殊符号，比如空格、$等
3）文件名统一使用小写字母，文件名使用英文名词命名，或英文简写

二、css规范

1、 代码风格

1）腾讯：css属性值需要用到引号时，统一使用单引号

1.selectors{ font-family:'\5FAE\8F6F\96C5\9ED1';}

2）为单个css选择器或新申明开启新行

1..home-count .hd,

2..content-title,

3..select-game-title .title{}

3）css属性书写顺序
建议遵循：布局定位属性 自身属性 文本属性 其他属性 CSS3属性

  • a 布局定位属性

display list-style position（相应的 top,right,bottom,left） float clear visibility /overflow

  • b 自身属性

width height margin padding border background

  • c 文本属性

color font text-decoration text-align vertical-align white- space break-word

  • d 其他（CSS3）

content cursor border-radius box-shadow text-shadow background:linear-gradient …

4）私有属性要写前面，标准写在后面

1.selectors{

2.    background: -webkit-linear-gradient(top, #ffffff 0%,#eff2f4 );

3.    background: -moz-linear-gradient(top, #ffffff 0%, #eff2f4 );

4.    background: -ms-linear-gradient(top, #ffffff 0%,#eff2f4 );

5.    background: -o-linear-gradient(top, #ffffff 0%,#eff2f4 );

6.    background: linear-gradient(to bottom, #ffffff 0%,#eff2f4 );

7.    filter: progid:DXImageTransform.Microsoft.gradient( startColorstr='#ffffff', endColorstr='#eff2f4',GradientType=0 );

8.}

5)删除css属性值为0的单位

1./* 不推荐 */ 

2.selectors{ margin:0px; padding:0px;}

3./* 推荐 */ 

4.selectors{ margin:0; padding:0;}

6) css属性 要使用Tab键缩进

1.selectors{

2.    height:  30px;

3.    width: 200px;

4.}

7) css属性与值写法（根据个人习惯选择），属性与值之间是否有空格

1.selectors{

2.    height : 30px;

3.    padding-bottom : 10px;

4.    border-bottom : 1px solid #858585;

5.    margin-bottom : 10px;

6.}
